HTML

At its heart, HTML is a language made up of elements, which can be applied to pieces of text to give them different meaning in a document (Is it a paragraph? Is it a bulleted list? Is it a part of a table?), structure a document into logical sections (Does it have a header? Three colums of content? A navigation menu?), and embed content such as images and videos into a page.

Elements consist of:

  • A tag name
  • Required or optional attributes
It can be easier to understand what is going on in your code if you have it nicely formatted. In this HTML each nested element is intented by four spaces more than the one it is sitting inside. It is up to you to choose the style of formatting (how many spaces for each level of indentation, for example).
